# account-kata
## Objectif

 - Déposer une somme sur un compte.
 - Retirer une somme sur un compte.
 - Avoir un historique des opérations.

## Installation

- Installer NodeJS.
- Cloner le projet.
- Faire ```npm install``` à la racine du projet.

## Lancer les tests

La commande ```npm test``` permet de lancer les tests avec ```jest```
